Firebase Notification Module Documentation

Overview

The Firebase Notification module for Odoo enables you to send push notifications to users using Firebase Cloud Messaging (FCM). With this module, you can easily configure and send notifications from your Odoo instance to targeted users or devices.

Features

  • Send push notifications to users through Firebase Cloud Messaging (FCM).
  • Target specific users or devices with personalized messages.
  • Easily configure the Firebase API credentials for seamless integration.

Installation

  1. Obtain the Firebase API credentials from the Firebase Console.
  2. Log in to your Odoo instance as an administrator.
  3. Go to the Apps menu and search for "Firebase Notification" module.
  4. Click on the module and then click on the "Install" button to install it.
  5. After installation, go to the "Settings" menu and search for "Firebase Configuration".
  6. Enter the Firebase API credentials obtained from step 1 into the provided fields.
  7. Save the configuration.

Configuration

  1. Go to the "Settings" menu in Odoo.
  2. Search for "Firebase Configuration" and click on it.
  3. Fill in the following information:
  • Server Key: Enter the Firebase Server Key obtained from the Firebase Console.
  • Sender ID: Enter the Firebase Sender ID obtained from the Firebase Console.
  1. Save the configuration.

Sending Notifications

  1. Go to the "Firebase Notifications" menu in Odoo.
  2. Click on the "Create" button to create a new notification.
  3. Fill in the following details:
  • Title: Enter the title of the notification.
  • Message: Enter the content of the notification.
  • Target: Select the target audience for the notification (specific users, user groups, or devices).
  • Additional Options: Provide any additional options for the notification (optional).
  1. Click on the "Send" button to send the notification.

Best Practices

  • Keep your notifications concise and relevant to engage users effectively.
  • Target your notifications to specific user groups or devices for personalized messaging.
  • Test your notifications thoroughly before sending them to a large audience.
  • Monitor the delivery and engagement rates of your notifications to optimize their impact.

Troubleshooting

  • Double-check your Firebase API credentials for accuracy.
  • Verify that your Odoo instance has a stable internet connection.
  • Ensure that the Firebase Cloud Messaging service is enabled for your project in the Firebase Console.
  • Review the Odoo server logs for any error messages related to the Firebase Notification module.
